Flow 2 - Byg din portfolio

Kod din portfolio (portfolio 1)

Opgaven lød på at at vi skulle udvikle vores portoflio i form af et website som vi udviklede ved hjælp af HTML5 og CSS3. Hvis du har lyst til at kigge nærmere på min første kodet portfolio kan du klikke på linket forneden samt kigge nærmere på min kodning ved at klikke på knappen "Github".


Læring i flow 2

  • Teknisk viden om internettet
  • Grundlæggende HTML 5 og CSS 3
  • Kendskab til en IDE (integrated development environment)
  • Optimering af billeder og grafik til web
  • Kendskab til layout principper
  • Versionsstyring via GitHub
  • Logisk navngivning og strukturering af filer, web projekt struktur FTP upload til eget domain
